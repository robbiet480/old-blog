My setup

I get asked this question quite a bit so here is my current computer setup:

  • I use a 27" Late 2009 Core i7 iMac (with 8GB RAM, 2TB drive) as my desktop and day-to-day development machine. It is hooked up to a 24 inch Samsung display.
  • My around town machine is a 17" Mid 2009 MacBook Pro, stock configuration.
  • Netbook #1 is a HP Mini 110-1025dx with Mac OS X 10.5.7.
  • Netbook #2 is a Google CR-48 Prototype running Chrome OS and Ubuntu.
  • I have a custom built HTPC with 2x2TB hard drives in a RAID 0 running Ubuntu 10.04.3 with a XBMC frontend
  • My current phone is an Apple iPhone 4, 32GB in black.
  • My current tablet is an Apple iPad 1, 32GB, 3G.
  • My tools of choice are Google Chrome, Coda, Textmate and Transmit.
  • I _always_ have a Terminal open on any computer.
  • My servers are hosted at Dreamhost, Linode, Hivelocity and SoftLayer.

This Site

I purchased robbie.io in 2011 and started it off with Tumblr. Today, it is automatically generated by Jekyll into a static site. Posts are in Markdown and the source resides on GitHub. Site is marked up with HTML5, styled with CSS3 and scripted with jQuery and JavaScript. All CSS and HTML syntax on this site has been validated. The comments are powered by Disqus. This theme is based on both Tom Preston-Werner and Tate Johnson's work with a little bit of the old Tumblr theme mixed it. If you find an error, please let me know.

This site is keyboard shortcut enabled:

  • h → home
  • o → about
  • l → lifestream
  • c → contact
